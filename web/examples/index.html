CxAgent Examples

This page provides access to various examples and test environments for the CxAgent chat widget.

Development Examples

  • Standalone Widget
    Demonstrates the chat widget itself as a standalone application.
  • Embed Demo (Dev)
    Demonstrates how to embed the chat widget in a website (development mode).
  • Embed Demo (Prod)
    Demonstrates how to embed the chat widget in a website.
  • Cross-Domain Test
    Simulates a production environment where the chat widget and backend servers are on different domains. Mimic production environment where the serving page at one domain embeds the widget script from another domain such as a CDN, which calls the agent service endpoint at yet another domain. Used for testing CORS configurations.

Testing Modes

Development Mode: Uses the Vite dev server to test React components and the embedding script.

Production Mode: Simulates a real-world scenario where components are on different domains.

For cross-domain testing, you'll need to run multiple servers:

  • Run Agent service endpoint for receiving messages: http://localhost:5678
  • Run Agent service endpoint for subscribing to response streams: http://localhost:5679
  • Make CxAgent Chat Widget Script available for embedding at a "CDN": http://localhost:3001
  • This examples page playing the role of an arbitrary Web server (e.g., http://localhost:3002)
